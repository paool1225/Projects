What On Earth Did You Just Do!?

I was the Producer for a final board game for my CTIN 488 class. This is a Social game where players get to know each other through deep discussions, but must also beware because each player has secret actions they must get the other players to complete. And the person with the most points wins.

The players navigate through the board where they usually draw and action card, discussion card, or swap out their actions either from the deck or another players hand.

Labarynth

A game I coded using HTML and CSS. it is meant to be a maze game where the player starts at the starting block and must navigate through the maze and find the green squares all the way to the finish, finding a red square means they must restart.

I used simple HTML and CSS to code this game and utilized the hover function and created multiple rows and columns to create the maze.

Roll For Dough-Ception

A board game I created with a group where we had certain mechanics and constrainst we had to follow. This is a cooking based game where the players must work together to complete their meal before the head chef completes their meal.

Our constraints were that it must be a player/multiple players vs game, there must be a bluffing/cheating mechanic, as well as a capture feature, and a dramatic element of cooking.

Race Against Thieves

The first Board game I created this semester with a group for CTIN 488. We had to create a mod of the game Up The River, where we took its core mechnics and twisted them and changed dramatic elements to create a new game that was similar but different enough to Up The River to be a separate game.

This game follows 4 thieves who are racing to be the first to make it to the jewel safe and back in order to win, but they must watch out for all the obstacles such as lasers, pressure plates, or security cameras, but don't worry because there is also a grappling hook that'll give you a boost
